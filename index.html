Notatio-Antiqua by dgj1983

Editor for GABC files, TeX files and LyTeX files (LilyPond and LaTeX)

This is intended to be a simple editor for writing gabc scores for gregorio and luaLaTeX. At the moment, it is in a beta state, for not everything works as it is expected. You are heartily invited to contribute to the existing code (in Qt (4.7.0 and up) and C++)

Dependencies

Qt >=4.7.0 GCC

Install

For source code installation, see readme file. The Win32-Setup can be run as every setup. Under Mac, mount the dmg image, drag the Application symbol to the applications folder and place the font and the template directory to the desired location. Sorry, there is no deb / rpm package yet.
